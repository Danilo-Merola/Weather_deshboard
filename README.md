# Weather Dashboard

Aplicação web para visualização de dados climáticos em tempo real, com previsão detalhada e interface moderna. O projeto consome APIs externas para fornecer informações atualizadas de forma dinâmica e responsiva.

---

## Funcionalidades

* Busca de cidades em tempo real
* Exibição do clima atual com sensação térmica
* Informações de vento (velocidade e direção)
* Dados de umidade e precipitação
* Previsão climática para 7 dias
* Visualização de temperatura por hora
* Interface responsiva e otimizada para diferentes dispositivos

---

## Tecnologias utilizadas

* HTML5
* CSS3 (layout moderno e animações)
* JavaScript (Vanilla)
* API Open-Meteo
* API de Geolocalização
